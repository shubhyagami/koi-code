About Java IDE

Java IDE v1.0

An IntelliJ-like web-based IDE for Java practice.

Built with Spring Boot, Thymeleaf, and Monaco Editor.

Features:

  • In-browser Java editing with syntax highlighting
  • Real-time compilation and execution
  • Project file management
  • IntelliJ-inspired dark theme
